# unity-validate

Entrega una validación manual clara para Unity del cambio actual.

Formato:
1. Escena a abrir
2. Pasos exactos a ejecutar
3. Qué observar en Game view
4. Qué observar en Scene view
5. Qué validar en NavMesh
6. Qué validar en selección/hover/outline/healthbar
7. Resultado esperado
8. Posibles fallas y cómo aislar si ocurren

No modificar código. Solo plan de validación.

This command will be available in chat with /unity-validate
